# A 股量化系统 v2.0 使用手册

## 1. 系统概述
本系统是一个基于 **Polymarket 预测市场** 与 **A 股微观资金** 的跨市场量化决策辅助系统。
核心逻辑：
- **宏观面**：利用 Polymarket 的概率变化预判全球宏观趋势（战争、加息、科技突破）。
- **微观面**：通过腾讯/东财 API 获取 A 股实时资金流向与板块异动。
- **融合决策**：将宏观概率映射为 A 股板块机会，结合 ATR 动态风控模型给出交易建议。

## 3. 快速使用

### 3.1 运行量化主程序
系统主入口，包含数据抓取、打分、风控计算和虚拟盘核对。
```bash
python3 quant_v2/main.py
```
**输出**：
- 终端打印实时市场状态与评分。
- 输出结构化 JSON (供 Agent 解析)。

### 3.2 独立运行数据采集
如果只需要获取行情数据，不运行评分逻辑：
```bash
python3 a_stock_data.py
```

## 4. Cron Job 自动化集成 (Hermes Agent)

推荐配合 Hermes Agent 的 `cronjob` 功能实现全自动早报与盘中监控。

### 4.1 每日早报 (08:00)
- **任务**：抓取隔夜外盘 + 早盘策略。
- **执行命令**：`python3 quant_v2/main.py`
- **Prompt 逻辑**：Agent 读取脚本输出的 JSON 评分，结合 Polymarket 数据生成完整报告。

### 4.2 盘中雷达 (10:00 / 14:30)
- **任务**：盘中异动监控 + 观察池追踪。
- **执行命令**：`python3 a_stock_data.py`
- **Prompt 逻辑**：Agent 抓取实时价格，检查观察池标的是否触发“止损”或“止盈”。

## 5. 核心算法说明

### 5.1 ATR 动态止损
不再使用固定百分比（如 5%）。系统根据股票当天的波动率（最高价 - 最低价）自动计算止损宽幅。
- **公式**：`Stop Loss = Price - (2 * Daily Range)`
- **优势**：防止高波动股票被轻易洗出，同时在低波动股票下跌时敏锐离场。

### 5.2 虚拟盘 (Paper Trading)
系统会自动将每日推荐标的存入 `quant_v2/quant_system.db`，并在次日运行时自动核对盈亏，实现自我进化。
