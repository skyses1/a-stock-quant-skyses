## 二、日胜率口径详解

### 2.1 日级涨跌分布

| 类别 | 天数 | 占比 | 说明 |
|------|------|------|------|
| **上涨日** | 130 天 | 44.2% | 当日持仓组合净值 > 前一日 |
| **下跌日** | 139 天 | 47.3% | 当日持仓组合净值 < 前一日 |
| **持平日** | 17 天 | 5.8% | 持仓但净值变化 = 0（如一字涨停/跌停无法成交，或涨跌恰好抵消） |
| **空仓日** | 8 天 | 2.7% | 无持仓（Kill Switch 触发后、或回测初期未建仓） |
| **合计** | **294 天** | 100.0% | |

### 2.2 日胜率计算公式

```
日胜率 = 上涨日 / (上涨日 + 下跌日) = 130 / (130 + 139) = 130 / 269 = 48.3%
```

> **口径说明**：
> - 分母排除持平日和空仓日，仅计算有实质涨跌的交易日
> - 若按总交易日计算：130 / 294 = 44.2%（v3 报告使用的口径）
> - **本报告采用排除法口径（48.3%）**，更能反映"有持仓且产生波动"时的胜率
> - 两种口径均在下方标注，避免歧义

| 口径 | 公式 | 结果 |
|------|------|------|
| 排除法（推荐） | 上涨日 / (上涨日+下跌日) | 48.3% |
| 总交易日法 | 上涨日 / 总交易日 | 44.2% |

## 六、失败归因

策略跑输等权基准（+3.68% vs +23.14%）且 62% 随机组合跑赢策略的原因：

1. **动量因子失效**：20 日动量在震荡市"追高杀跌"，买入时已涨过一轮，买入后即回落
2. **换手成本侵蚀**：58 次调仓产生 69,787 元成本（占本金 6.98%），直接吞噬大部分超额收益
3. **股票池偏斜**：78 只新浪排序靠前的小盘股，波动率天然偏高（年化 27.10%），高波动+高动量组合在反转时跌幅更大
4. **波动率未纳入权重**：选股未考虑波动率调整，导致高波股票权重过大
5. **涨跌停拦截影响有限**：仅 1 次一字涨停+2 次零成交量被拒，非主因
